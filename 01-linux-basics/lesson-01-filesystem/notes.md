# Lesson 01 - Linux File System

## Key paths

- `/` is the root of the Linux file system.
- `/home/phantom` is my Linux user home directory.
- `~` is a shortcut for `/home/phantom`.
- `/mnt/c` is the Windows C drive mounted inside Linux.
- `/etc` contains system and application configuration files.
- `/var/log` contains log files.
- `/tmp` is used for temporary files.

## Important rule

I should keep my DevOps projects inside the Linux file system, for example `~/devops-labs`, not inside `/mnt/c/Users/...`.
