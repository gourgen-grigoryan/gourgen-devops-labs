# Lesson 03 - File Operations

## Commands learned

- `touch` creates an empty file.
- `mkdir` creates a directory.
- `mkdir -p` creates parent directories if needed.
- `cp` copies files.
- `cp -r` copies directories recursively.
- `mv` moves or renames files and directories.
- `rm` removes files.
- `rmdir` removes empty directories.
- `cat` prints file content.
- `echo "text" > file` writes text to a file and overwrites existing content.
- `echo "text" >> file` appends text to the end of a file.

## Safety rule

Before using delete commands like `rm` or especially `rm -rf`, I should always check my location with `pwd` and inspect files with `ls` or `tree`.
